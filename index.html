django-featured-objects

Django app for making any object featured

Download this project as a .zip file Download this project as a tar.gz file

Install

  • pip install django-featured-objects

Configuration

    # add to settings.py
    INSTALLED_APPS += ('featured',)
    FEATURABLE_MODELS = (
        ('app_label', 'model_name'), 
        ('app_label', 'another_model_name')
    )

Use

Go to admin site and add objects by id and content type to featured app.

Contribute

Fork and pull request, will be added to pypi soon.
